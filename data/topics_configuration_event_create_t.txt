 Create an Event for the Content Definition Author the content definition by creating an event. Let's create a Create Event for our ToDo content definition. This event will be triggered when a ToDo entity is created. Click next to the Events. Content Definition Events The Create Event is displayed. Create Event Enter the Event name. By default, the Extends from event content type value is selected. Select Static checkbox if you do not want the event to be instantiated. Static content will not have any associated action. Enter the Description. The description is mandatory in all the places for documentation purpose. The information provided in this field will generate documentation for the content definition action. Click next to the Event Payload to define the event payload information. The Event Payload field are displayed. Event Payload Click Create. The content definition lists the created event. Created Content Definition Event In this example, we have created a Create event. Now, we can create Delete and Modified events.
