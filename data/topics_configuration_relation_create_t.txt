 Create a Relation for the Content Definition Author the content definition by creating a relation. Let's create a relation between ToDos and Notes in our application. Click next to the Relations. Content Definition Relations The Create Relation is displayed. Create Relation Enter the Relation name. Select the Content type of related content from the drop-down list. Select the From cardinality [from this content to related content] from the drop-down list. From Cardinality Select the To cardinality [from this content to related content] from the drop-down list.
